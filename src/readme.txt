Для тестирования в папке src должна быть папка с названием resources.
В ней должны быть:
1) файл базы данных Sqlite
   с названием bot_persistence.db
2) файл с токеном вашего
   бота для тестирования с названием telegram_bot_token,
   который должен содержать одну строку, являющуюся
   непосредственно токеном вашего бота.

Для экспериментов, создайте файл launch_sanbox.py и
папку sandbox (эти имена есть в .gitignore, поэтому случайно вы
их не поместите в коммит).

Чтобы всё работало, нужно запускать скрипты в директории src,
иначе импорт модулей не будет работать.

Старайтесь придерживаться PEP 8, самое главное:
НЕ ПИШИТЕ СТРОКИ ДЛИННЕЕ 80 СИМВОЛОВ.
